# Proj-MC


The server file is in the Backend Module

In the server, a DB connection is made.

Sockets are used to establish a connection ensuring real-time chat and at the same time it's also updating the Database.

For the front end, A simple HTML page is created displaying the real-time chat, happening.
Also If one user from the other device sends a message then it will appear on all users and will be saved in the database too.

On fetching the first time, all the messages stored in databases will be visible to users.
All chatting is done in real-time and the connection is made using a socket, ensuring reliability and stability.

Tech Used- Node.Js, MongoDB, HTML, Socket.IO
